# vNEXT - Unreleased Features

This directory contains documentation for features currently under development.

## Upcoming Features

Documentation for new features will appear here during development. Once features are released and stable, their documentation will be integrated into the main docs.

## Contributing

When adding a new feature:
1. Create documentation here in `docs/versions/vNEXT/`
2. Reference the issue or PR number
3. Include usage examples and migration notes if applicable
4. Move to main docs when feature is released
